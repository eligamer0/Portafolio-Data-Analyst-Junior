# Portafolio Profesional de Data Analyst

Portafolio moderno y responsive para Eliezer Rosario, construido con React, Vite, React Router DOM y CSS Modules.

## Caracteristicas

- Modo oscuro profesional
- Navegacion responsive con React Router
- Lazy loading por pagina
- Componentes reutilizables
- Casos de estudio para dashboards de ventas e inventario
- Barras visuales de habilidades
- Proceso de analisis de datos
- Formulario de contacto
- SEO basico en `index.html`

## Ejecutar el proyecto

```bash
npm install
npm run dev
```

Luego abre:

```text
http://localhost:5173
```

## Crear build de produccion

```bash
npm run build
```
